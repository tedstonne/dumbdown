# Dumbdown

Browser extension for AI-powered TL;DR summaries.

## Commands

- **Build**: `npm run build` (both browsers)
- **Build Chrome**: `npm run build:chrome`
- **Build Firefox**: `npm run build:firefox`
- **Package**: `npm run package` (both browsers)
- **Package Chrome**: `npm run package:chrome`
- **Package Firefox**: `npm run package:firefox`
- **Test**: `npm test`
- **Clean**: `npm run clean`
