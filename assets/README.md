# Store Assets

## How to Create Screenshots

1. Open `promo.html` in browser
2. Screenshot at 1280x800
3. Save as `promo.png`

## Source Files
- `promo.html` - Promo page (screenshot this)
- `icon-base.png` - 512x512 app icon
- `icon-preview.svg` - Vector source for icon

## Brand Colors
- Primary: `#5CE0CD` (teal)
- Background: `#1a1a1a` (near-black)
